# AI & Machine Learning Projects

This repository contains a collection of academic and practical projects developed during studies and practice in **Machine Learning, Computer Vision, Reinforcement Learning, and Software Engineering**.  
Each folder corresponds to an independent project focused on a specific topic, framework, or methodology.

### C++_project
A project developed in **C++** for the creation of a contact rubric management system. 
This project focuses on core programming concepts such as:
- object-oriented programming,
- memory management,
- algorithms and data structures,

---

### Linear_Regression_project (Sklearn)
A project dedicated to **linear regression** on a real-world housing dataset, including:
- EDA and preprocessing pipeline,
- implementation of regularization (L1 and L2) and polynomial features,
- training and validation pipeline using Cross Validation,

---

### ML_Classification_project (Sklearn)
A **machine learning classification project**, covering:
- classical k-NN classification algorithm on a real world dataset to classify fruit types.
- feature engineering,
- model comparison and performance evaluation.

---

### Computer_Vision_TF_project (Tensorflow)
A **computer vision project implemented using TensorFlow** on the CIFAR dataset, covering topics such as:
- image preprocessing and data augmentation,
- convolutional neural networks (CNNs),
- model training and evaluation with TensorFlow/Keras.

---

### Computer_Vision_PT_project (Pytorch)
A **PyTorch-based computer vision project** on the Food Classification dataset, focusing on:
- convolutional neural networks,
- training pipelines in PyTorch,
- experimentation with different architectures and hyperparameters.

---

### Computer_Vision_PT_project2 (Pytorch)
A second **PyTorch computer vision project** on a Flower Classification dataset, including:
- transfer learning,
- modern architectures (e.g. ResNet, EfficientNet),
- comparison of data augmentation techniques.

---

### Reinforcement_Learning_project (Pytorch)
A project focused on **reinforcement learning** in a CyberSecurity setting, including:
- classical SARSA implemented via linear function approximation,
- Q-learning implementation via NNs (DQN),
- reward-shaping

---

### MLOps_project (FASTAPI, DOCKER, GIT, etc...)
A project, hosted in this separated [https://github.com/lucabadiali/MLOPS_Project](repo), for the creation of a REST API application for sentiment analysis, including:
- FASTAPI use for API creation,
- application hosting on HuggingFace,
- CI/CD via Git actions,
- Docker for application deployment,
- model monitoring via Prometheus/Grafana,
